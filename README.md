# Platformer2DPack
Pack of components for 2D platformer-like game.
## Installation via git URL
1. Open **Unity**
2. Go to **Window → Package Manager**
3. Click **+** → **Add package from Git URL**
4. Paste the following URL:<br>
`https://github.com/Tomas-618/Platformer2DPack.git?path=Assets/Source/Platformer2DPackage#1.0.0`
